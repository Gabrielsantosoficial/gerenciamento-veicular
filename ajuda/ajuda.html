Ajuda

Para registrar uma nova viagem, siga os passos:

  1. Selecione o veículo desejado na tela inicial
  2. Clique em "Informar Viagem"
  3. Preencha os dados necessários
  4. Confirme as informações
Como reportar um problema?
Para reportar um problema, selecione o veículo na tela inicial, clique em "Reportar Problema" e descreva o problema encontrado.
